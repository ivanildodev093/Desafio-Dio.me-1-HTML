Historia do HTML

(voltar)

A história do HTML foi inventado em 1990 por um cientista chamado Tim Berners-Lee. A finalidade inicial era a de tornar possível o acesso e a troca de informações e de documentação de pesquisas, entre cientistas de diferentes universidades. O projeto inicial tornou-se um sucesso jamais imaginado por Tim Berners-Lee. Ao inventar o HTML ele lançou as fundações da Internet tal como a conhecemos atualmente.

Estrutura básica do HTML

(Voltar)

Para desenvolvermos sites em HTML usamos a ferramenta chamada Visual Code, atravez das "tags" modificamos todo o corpo e estrutura do site!!

Um site em HTML e composto por Head(cabeça) e Body(corpo)

  • Head(cabeça): É a primeira informação que chega ao cliente antes de carregar o site ela nomeia a aba do navegador!

  • Body(corpo):Contem todo o conteudo do site informaçoes links etc...

Tags usuais

(voltar)

As tags são usadas para informar ao navegador a estrutura do site. Ou seja: quando se escreve um código em HTML, as tags serão interpretadas pelo navegador, produzindo assim a estrutura e o conteúdo visual da página. Elas sempre estarão dentro dos sinais de chevron, iniciando (<>) ou finalizando ()

Atributos

(Voltar)

Os atributos são usados para personalizar as tags alterando sua estrutura ou funcionalidade. Do mesmo jeito os atributos são usados para atribuir uma classe ou ID a um elemento. A maioria das tags tem suas próprias propriedades. No entanto, alguns atributos genéricos podem ser usados na maioria das tags HTML

Listas ordenadas e não ordenadas

(Voltar)

Existem dois tipos de listas, as ordenadas <'ol'> e as não ordenadas <'/ul'> e detro dessas listas existem as filhas <'li'> e podemos fazer variaçoes dentro de cada uma das listas !!
